getWalletByUUID()

UserWalletsHelper.php : 131

Retrieves the wallet address for a given UUID and coin ID.

public static getWalletByUUID(string $uuid[, string $coin_id = '' ]) : string|false

If the UUID belongs to a user, the wallet is retrieved from the user's Fireblocks vault. If the UUID belongs to a product, the wallet is retrieved from the product's corresponding Fireblocks vault. If no wallet is found, a new one is created. If the coin ID is not provided, it is retrieved from the application settings.

Parameters
$uuid : string

The UUID of the user or product.

$coin_id : string = ''

The ID of the coin. Optional.

Return values
string|false —

The wallet address or false if an error occurs.
